Currency Exchange Rate

Overview!

This is conversion module between two currencies. This module use, "https://api.exchangerate-api.com" site's API to get currency rates. Here user will get updated exchange rate with desire amount.

Exchange Menu and Window!

User need to go Settings>Technical menu>Currency Exchange Rate menu. Here User will get new screen for exchange rate comparison. User need to set amount and two currency for comparison.

Convert!

When user set amount and two currency then need to click Convert button. The result value will be show below. Here current conversion rate and conversion amount will be visible.

Switch The Currency!

User can switch the currencies by clicking Switch button. Here currencies are switch and switched conversion value will be visible.

Clear!

User can clear all values for new comparison with Clear button.
